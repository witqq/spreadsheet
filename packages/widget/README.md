# @witqq/spreadsheet-widget

> Embeddable spreadsheet widget — single-file IIFE/UMD bundle

[![npm version](https://img.shields.io/npm/v/@witqq/spreadsheet-widget.svg)](https://www.npmjs.com/package/@witqq/spreadsheet-widget)
[![license](https://img.shields.io/npm/l/@witqq/spreadsheet-widget.svg)](https://github.com/witqq/spreadsheet/blob/master/LICENSE)

## Installation

```bash
npm install @witqq/spreadsheet-widget
```

Requires [`@witqq/spreadsheet`](https://www.npmjs.com/package/@witqq/spreadsheet) (installed automatically).
