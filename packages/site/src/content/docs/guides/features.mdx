@witqq/spreadsheet is a canvas-based spreadsheet engine for the web. This page maps every capability to its documentation.

## Core Engine

| Feature | Description | Docs |
|---|---|---|
| Canvas 2D Rendering | Layer-based pipeline, 60 FPS at 100K+ rows | [Rendering](/concepts/rendering/) |
| Data Model | Sparse CellStore, RowStore, ColStore, StylePool | [Data Model](/concepts/data-model/) |
| Event System | Typed EventBus with 36 events | [Events](/concepts/events/) |
| Themes | Light/dark themes, runtime switching, full customization | [Themes](/concepts/themes/) |
| Architecture | Zero-dependency core, plugin system, framework wrappers | [Architecture](/concepts/architecture/) |
