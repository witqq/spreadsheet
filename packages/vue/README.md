# @witqq/spreadsheet-vue

> Vue 3 wrapper for witqq Canvas spreadsheet engine

[![npm version](https://img.shields.io/npm/v/@witqq/spreadsheet-vue.svg)](https://www.npmjs.com/package/@witqq/spreadsheet-vue)
[![license](https://img.shields.io/npm/l/@witqq/spreadsheet-vue.svg)](https://github.com/witqq/spreadsheet/blob/master/LICENSE)

## Installation

```bash
npm install @witqq/spreadsheet-vue
```

**Peer dependencies:**

- `vue` ^3.3.0

Requires [`@witqq/spreadsheet`](https://www.npmjs.com/package/@witqq/spreadsheet) (installed automatically).
