# @witqq/spreadsheet-angular

> Angular wrapper for witqq Canvas spreadsheet engine

[![npm version](https://img.shields.io/npm/v/@witqq/spreadsheet-angular.svg)](https://www.npmjs.com/package/@witqq/spreadsheet-angular)
[![license](https://img.shields.io/npm/l/@witqq/spreadsheet-angular.svg)](https://github.com/witqq/spreadsheet/blob/master/LICENSE)

## Installation

```bash
npm install @witqq/spreadsheet-angular
```

**Peer dependencies:**

- `@angular/core` >=17.0.0
- `@angular/common` >=17.0.0

Requires [`@witqq/spreadsheet`](https://www.npmjs.com/package/@witqq/spreadsheet) (installed automatically).
